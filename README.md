# 3Trium | Desenvolvimento Web de Alto Nível 🚀

Landing page institucional desenvolvida para a **3Trium**, focada em apresentar soluções de elite para presença digital, como sites institucionais, landing pages e sistemas web customizados.

## 🛠️ Tecnologias Utilizadas

* **HTML5 & CSS3**: Estrutura e estilização semântica.
* **Tailwind CSS**: Framework utilitário para um design responsivo e moderno.
* **Swiper.js**: Slider interativo de alta performance para a seção principal.
* **AOS (Animate On Scroll)**: Biblioteca para animações fluidas durante a rolagem.
* **Font Awesome**: Ícones vetoriais profissionais.
* **Google Fonts**: Tipografia "Inter" para máxima legibilidade.

## ✨ Diferenciais do Projeto

* 🎯 **Design Exclusivo**: Focado em branding com gradientes roxos e estética premium.
* 📱 **Totalmente Responsivo**: Adaptado para qualquer tamanho de tela (Mobile, Tablet e Desktop).
* 🎬 **Seção de Vídeo Local**: Implementação de vídeo nativo otimizado para apresentação de diferenciais.
* ⚡ **Performance**: Código limpo e carregamento rápido utilizando bibliotecas via CDN.
* 🖱️ **Interatividade**: Efeitos de hover personalizados e contornos destacados nos cards de serviços.
